## Installation

### Dependencies

* `rich`
* `distro`
* `screeninfo`
* `psutil`

### Steps

1.  Clone repo:

    ```bash
    git clone https://github.com/cihaneray/TorchFetch.git
    ```

2.  Go to the directory:

    ```bash
    cd TorchFetch
    ```

3.  Install dependencies:
    
    ```bash
    pip3 install -r requirements
    ```
    or
    ```bash
    pip3 install rich screeninfo distro psutil
    ```
    
4. Running:

    ```bash
   python3 TorchFetch.py
    ```
